# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in cimishi, please report it responsibly.

**Do not open a public issue.** Instead, send an email to the maintainers or use [GitHub's private vulnerability reporting](https://github.com/TopSwagCode/cimishi/security/advisories/new).

Please include:
- A description of the vulnerability
- Steps to reproduce the issue
- The potential impact
- Any suggested fix (if applicable)

We will acknowledge your report within 48 hours and aim to release a fix promptly.
